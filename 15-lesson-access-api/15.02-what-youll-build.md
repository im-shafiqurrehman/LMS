# What You Will Build

| Endpoint | Who | What |
|----------|-----|------|
| GET /api/learn/:courseId/lessons | enrolled student | Full lesson list with video URLs |
| GET /api/learn/:courseId/lessons/:lessonId | enrolled student | Single lesson with video URL |
| POST /api/learn/progress | enrolled student | Mark a lesson complete |
| GET /api/learn/:courseId/progress | enrolled student | Progress data for a course |
| POST /api/learn/certificates/:courseId | enrolled student | Issue certificate (if complete) |
| GET /api/learn/certificates | enrolled student | List my certificates |
| GET /api/learn/:courseId/questions | enrolled student | List Q&A for a course |
| POST /api/learn/:courseId/questions | enrolled student | Post a question |
| POST /api/learn/questions/:qId/answers | instructor | Answer a question |

All /api/learn routes require authentication and enrollment verification.
