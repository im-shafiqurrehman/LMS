# Issue Certificate

POST /api/learn/certificates/:courseId — issues a certificate if all lessons are complete.

```js
async function issueCertificate(studentId, courseId) {
  const enrollment = await requireEnrollment(studentId, courseId);

  // Idempotent: return existing certificate if already issued
  const existing = await Certificate.findOne({ enrollmentId: enrollment._id });
  if (existing) return existing;

  // Verify completion
  const [totalLessons, completedCount] = await Promise.all([
    Lesson.countDocuments({ courseId }),
    ProgressRecord.countDocuments({ enrollmentId: enrollment._id }),
  ]);

  if (completedCount < totalLessons) {
    throw new AppError('Complete all lessons before requesting a certificate', 422);
  }

  return Certificate.create({
    enrollmentId: enrollment._id,
    issuedAt:     new Date(),
    // pdfUrl: will be set after PDF generation (bar-raiser)
  });
}
```

✅ Calling issueCertificate twice returns the same certificate (idempotent).
✅ Calling with incomplete progress returns 422.
