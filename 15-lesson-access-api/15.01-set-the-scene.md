# Lesson Access API — Set the Scene

Only enrolled students can access lesson video URLs. This chapter builds the enrollment guard at the API level, the video streaming endpoint, progress tracking, certificate issuance, and Q&A endpoints.

By the end:
- GET /api/learn/:courseId/lessons returns the lesson list with video URLs (for enrolled students)
- GET /api/learn/:courseId/lessons/:lessonId returns the lesson with its video URL (verified enrollment)
- POST /api/learn/progress marks a lesson complete
- GET /api/learn/:courseId/progress returns the student's progress on a course
- POST /api/learn/certificates/:courseId issues a certificate when all lessons are complete
- Q&A: list questions, post question, post answer
