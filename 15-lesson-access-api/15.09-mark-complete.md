# Mark Lesson Complete

POST /api/learn/progress marks a lesson as complete for the authenticated student.

```js
async function markLessonComplete(studentId, courseId, lessonId) {
  const enrollment = await requireEnrollment(studentId, courseId);

  // Idempotent: upsert avoids error on duplicate completion
  await ProgressRecord.findOneAndUpdate(
    { enrollmentId: enrollment._id, lessonId },
    { completedAt: new Date() },
    { upsert: true }
  );

  // Check if all lessons are now complete
  const [totalLessons, completedLessons] = await Promise.all([
    Lesson.countDocuments({ courseId }),
    ProgressRecord.countDocuments({ enrollmentId: enrollment._id }),
  ]);

  return {
    completed: true,
    courseCompleted: completedLessons >= totalLessons,
  };
}
```

✅ Marking the same lesson complete twice has no side effects (idempotent).
✅ courseCompleted: true when all lessons are marked.
