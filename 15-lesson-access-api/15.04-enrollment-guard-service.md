# Enrollment Guard — Service Level

Every /api/learn endpoint must verify enrollment before returning protected data. Create a reusable guard in the service:

```js
// src/modules/learn/learn.service.js — helper
async function requireEnrollment(studentId, courseId) {
  const enrollment = await Enrollment.findOne({ studentId, courseId });
  if (!enrollment) throw new AppError('Not enrolled in this course', 403);
  return enrollment;
}
```

Every service function in this module starts with:
```js
const enrollment = await requireEnrollment(req.user.id, courseId);
```

This separates the access check from the data retrieval — cleaner than embedding the check in each query.
