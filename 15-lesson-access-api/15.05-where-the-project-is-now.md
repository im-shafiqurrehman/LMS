# Where the Project Is Now

Create the module scaffold:

```bash
mkdir server/src/modules/learn
touch server/src/modules/learn/learn.routes.js
touch server/src/modules/learn/learn.service.js
```

Mount in app.js:
```js
import { learnRouter } from './modules/learn/learn.routes.js';
app.use('/api/learn', authenticate, learnRouter);  // all routes require login
```

By applying authenticate at the router level, the enrollment check in the service handles the rest. A logged-in non-enrolled student gets 403 from the service; an unauthenticated request gets 401 before reaching the service.
