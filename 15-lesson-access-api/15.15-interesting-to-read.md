# Interesting to Read

**1. S3 Presigned URLs**
Search "AWS S3 presigned URL Node.js SDK v3" — the modern SDK generates short-lived signed URLs without a proxy server. This is the production video URL pattern.

**2. Video streaming and Range requests**
Search "HTTP Range header video streaming Node.js" — if you serve video from your own server (not a CDN), you need to handle Range headers so browsers can seek in the video.

**3. PDF generation in Node.js**
Search "pdfkit Node.js tutorial" or "puppeteer generate PDF" — both approaches are used in production certificate systems.
