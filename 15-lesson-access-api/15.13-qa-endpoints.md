# Q&A Endpoints

Three Q&A endpoints for the lesson player:

**List questions for a course:**
```js
async function listQuestions(courseId, page = 1) {
  return Question.find({ courseId })
    .populate('userId', 'email')
    .sort({ createdAt: -1 })
    .limit(20);
}
```

**Post a question:**
```js
async function postQuestion(courseId, userId, content) {
  await requireEnrollment(userId, courseId); // must be enrolled to ask
  return Question.create({ courseId, userId, content });
}
```

**Post an answer (instructor only):**
```js
async function postAnswer(questionId, instructorId, content) {
  const question = await Question.findById(questionId);
  if (!question) throw new AppError('Question not found', 404);
  return Answer.create({ questionId, instructorId, content });
}
```

Mount the answer route with authorize('instructor', 'admin') middleware.
