# The ProgressRecord and Certificate Schemas

**ProgressRecord** tracks when a student completes a lesson within an enrollment:

```js
// src/models/progressRecord.model.js — shape only
import mongoose from 'mongoose';

const progressRecordSchema = new mongoose.Schema(
  {
    enrollmentId: {
      type:     mongoose.Schema.Types.ObjectId,
      ref:      'Enrollment',
      required: true,
    },
    lessonId: {
      type:     mongoose.Schema.Types.ObjectId,
      ref:      'Lesson',
      required: true,
    },
    completedAt: {
      type:    Date,
      default: Date.now,
    },
  },
  { timestamps: true }
);

progressRecordSchema.index({ enrollmentId: 1, lessonId: 1 }, { unique: true });

export const ProgressRecord = mongoose.model('ProgressRecord', progressRecordSchema);
```

The compound unique index prevents marking the same lesson complete twice.

**Certificate** is issued when a student completes all lessons:

```js
// src/models/certificate.model.js — shape only
import mongoose from 'mongoose';

const certificateSchema = new mongoose.Schema(
  {
    enrollmentId: {
      type:     mongoose.Schema.Types.ObjectId,
      ref:      'Enrollment',
      required: true,
      unique:   true,
    },
    pdfUrl:   { type: String, default: null },
    issuedAt: { type: Date, default: Date.now },
  },
  { timestamps: true }
);

export const Certificate = mongoose.model('Certificate', certificateSchema);
```

unique: true on enrollmentId ensures one certificate per enrollment.
