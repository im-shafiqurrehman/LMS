# Interesting to Read

**1. Redis data structures**
Search "Redis data structures tutorial" — Redis is not just a key-value store. Understanding hashes, sorted sets, and pub/sub opens architectural possibilities beyond simple caching.

**2. Distributed rate limiting with Redis**
Search "express-rate-limit Redis store" — the official plugin stores counters in Redis so rate limits work across multiple server processes.

**3. MongoDB Atlas Search relevance scoring**
Search "MongoDB Atlas Search relevance scoring" — Atlas Search returns a search score for each result that can be used for ranking.
