# Where the Project Is Now

Before building:

1. Redis should be running via Docker (from chapter 02). Verify: docker compose ps
2. MongoDB Atlas connection string should be in DATABASE_URL (if using Atlas for development)
3. Install dependencies:

```bash
cd server
npm install ioredis express-rate-limit
```

4. Add to .env:
```
REDIS_URL=redis://localhost:6379
```

5. Remove .optional() from REDIS_URL in config/index.js — it is now required
