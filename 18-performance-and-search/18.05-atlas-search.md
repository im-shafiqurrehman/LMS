# MongoDB Atlas Search

Atlas Search is a full-text search layer built on top of MongoDB Atlas. It uses Apache Lucene under the hood — the same engine that powers Elasticsearch.

To use it:
1. Create a search index in the MongoDB Atlas UI (Clusters → Search → Create Index)
2. Define the index fields: title (string), description (string)
3. Use the $search aggregation stage in your query:

```js
const courses = await Course.aggregate([
  { $search: { index: 'default', text: { query: q, path: ['title', 'description'], fuzzy: {} } } },
  { $match: { status: 'published' } },
  { $limit: 12 },
]);
```

fuzzy: {} enables typo tolerance — searching "reactt" matches "react".
