# Performance and Search — Set the Scene

EduFlow now has all its features. This chapter makes it fast and searchable:
- Redis caching for the course catalogue (most-read, rarely-changing)
- MongoDB Atlas Search for full-text keyword search (replacing the regex)
- Rate limiting on the auth endpoints (preventing brute-force attacks)
- Notifications (email and/or real-time)

These are production concerns that separate a working prototype from a production-ready system.
