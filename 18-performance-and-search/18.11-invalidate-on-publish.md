# Invalidate Cache on Publish

In the admin service (chapter 19), when a course is approved (status → published), invalidate the catalogue cache:

```js
async function approveCourse(courseId, adminId) {
  const course = await Course.findByIdAndUpdate(
    courseId,
    { status: 'published' },
    { new: true }
  );
  // Invalidate all catalogue cache keys
  const keys = await redis.keys('courses:list:*');
  if (keys.length > 0) await redis.del(keys);
  return course;
}
```

Without this, newly published courses would not appear in the catalogue for up to 5 minutes after approval.
