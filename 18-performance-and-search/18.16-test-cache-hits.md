**Test 2 — Rate limit:**
```bash

for i in $(seq 1 11); do

    -H 'Content-Type: application/json' \
    -d '{"email":"bad@test.com","password":"wrong"}'
done
# 11th request: { "error": "Too many requests..." }
```
