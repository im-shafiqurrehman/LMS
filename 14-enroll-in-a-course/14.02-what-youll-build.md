# What You Will Build

Four pieces:

1. **Enrollment check** on the course detail page — show Enroll or Go to Course based on enrollment status
2. **Checkout page** — Stripe Elements payment form
3. **Payment result handler** — after payment completes, the frontend polls for the enrollment (webhook may take 1-2 seconds)
4. **My Learning page** — list of the student's enrolled courses

Install Stripe's frontend SDK:
```bash
cd client
npm install @stripe/react-stripe-js @stripe/stripe-js
```

Add to client/.env.local:
```
NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY=pk_test_your_key
```
