# Progress Tracker

Create components/ProgressBar.tsx:

```tsx
export function ProgressBar({ percentage }: { percentage: number }) {
  return (
    <div style={{ background: '#eee', borderRadius: 4, overflow: 'hidden', height: 8 }}>
      <div
        style={{ background: '#28a745', width: `${percentage}%`, height: '100%', transition: 'width 0.3s' }}
      />
    </div>
  );
}
```

Show it in the lesson player layout above the lesson list. The percentage comes from GET /api/learn/:courseId/progress on mount and updates after each Mark Complete action.
