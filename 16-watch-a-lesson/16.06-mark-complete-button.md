# Mark Complete Button

The Mark Complete button calls POST /api/learn/progress and updates local state:

```tsx
async function handleMarkComplete() {
  const { data } = await api.post('/api/learn/progress', { courseId, lessonId });
  setCompletedIds(prev => [...prev, lessonId]);
  if (data.courseCompleted) {
    setShowCertificatePrompt(true);
  }
}
```

Show the button only if the lesson is not already marked complete:

```tsx
{!completedIds.includes(currentLessonId) && (
  <button onClick={handleMarkComplete}>Mark as Complete</button>
)}
{completedIds.includes(currentLessonId) && (
  <span style={{ color: '#28a745' }}>✓ Completed</span>
)}
```
