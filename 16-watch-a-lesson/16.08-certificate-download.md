# Certificate Download

After issuance, offer the certificate as a download:

```tsx
async function handleIssueCertificate() {
  const { data } = await api.post(`/api/learn/certificates/${courseId}`);
  setCertificate(data);
}

// Render:
{certificate && (
  certificate.pdfUrl
    ? <a href={certificate.pdfUrl} download>Download Certificate (PDF)</a>
    : <p>Certificate issued on {new Date(certificate.issuedAt).toLocaleDateString()}. PDF generation is pending.</p>
)}
```

When pdfUrl is null (the stub), show the issue date without a download link.
