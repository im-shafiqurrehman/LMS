# Lesson Player Component

Create components/LessonPlayer.tsx. For now, a simple HTML5 video player:

```tsx
export function LessonPlayer({ videoUrl, title }: { videoUrl: string; title: string }) {
  return (
    <div style={{ width: '100%' }}>
      <h1>{title}</h1>
      <video
        src={videoUrl}
        controls
        style={{ width: '100%', maxHeight: '60vh', background: '#000' }}
      />
    </div>
  );
}
```

The bar-raiser version uses a professional player like Video.js or react-player with quality selection, subtitle tracks, and playback speed controls.
