# Verify the Full Setup

Run through each verification step before declaring this chapter done.

**1. Config fail-fast:**

```
# Temporarily remove DATABASE_URL from .env
# Rename it: DATABASE_URL_DISABLED=...
npm run dev
```

Expected: the server refuses to start and prints:

```
Invalid environment config:
{ DATABASE_URL: { _errors: ['DATABASE_URL is required'] } }
```

Restore DATABASE_URL.

**2. Database connection:**

```
npm run dev
```

Expected console (in order):
```
MongoDB connected: localhost
EduFlow running on port 3000 [development]
```

**3. Health check:**

```
curl http://localhost:3000/api/health
```

Expected: { "status": "ok", "timestamp": "..." }

**4. Error handler:**

Add a test route to app.js temporarily:

```js
app.get('/api/test-error', (req, res, next) => {
  next(new Error('test error'));
});
```

```
curl http://localhost:3000/api/test-error
```

Expected: { "error": "test error", "stack": "..." } (stack only in development)

Remove the test route.

**5. Git status:**

```
git status
```

Expected: only source files staged — no .env, no .env.local.
