# Where the Project Is Now

Before building:

1. Install Stripe: npm install stripe
2. Add to server/.env:
   STRIPE_SECRET_KEY=sk_test_your_stripe_test_key
   STRIPE_WEBHOOK_SECRET=whsec_your_webhook_signing_secret
3. Remove .optional() from these fields in config/index.js (they are now required)
4. Create the module scaffold:

```bash
mkdir server/src/modules/enrollments
touch server/src/modules/enrollments/enrollments.routes.js
touch server/src/modules/enrollments/enrollments.service.js
```

Mount in app.js:
```js
import { enrollmentsRouter } from './modules/enrollments/enrollments.routes.js';
app.use('/api/enrollments', enrollmentsRouter);
```

The webhook route needs raw body access (Stripe verifies the signature against the raw body — JSON.parse breaks it). Express must be configured to preserve raw body for the webhook endpoint specifically.
