# Chapter 13 — Definition of Done

- [ ] POST /api/enrollments/intent creates a Stripe PaymentIntent and returns the client secret
- [ ] POST /api/enrollments/webhook verifies the Stripe signature; creates Enrollment on payment_intent.succeeded
- [ ] Calling the webhook twice with the same paymentIntentId only creates one Enrollment
- [ ] GET /api/enrollments/mine returns the student's enrollments with course data
- [ ] GET /api/enrollments/check/:courseId returns { enrolled: true/false }
- [ ] amountPaid on Enrollment reflects the course price at time of payment
- [ ] Stripe CLI or stripe trigger tested locally
- [ ] git commit: "feat(enrollments): Stripe payment intent, webhook, idempotent enrollment"

All boxes ticked? Continue to chapter 14 — Enroll in a Course (Frontend).
