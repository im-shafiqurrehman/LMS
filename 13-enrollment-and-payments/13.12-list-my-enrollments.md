# List My Enrollments

GET /api/enrollments/mine — returns all of the student's enrollments with course data.

```js
async function listMyEnrollments(studentId) {
  return Enrollment.find({ studentId })
    .populate('courseId', 'title slug coverImageUrl totalLessons totalDuration')
    .sort({ enrolledAt: -1 });
}
```

✅ Returns the course title and cover image so the frontend can display the enrollment list without additional requests.
✅ Student only sees their own enrollments (the query filters by studentId from req.user.id).
