# Test the Enrollment API

```bash
BASE=http://localhost:3000

# 1. Login as student
STUDENT_TOKEN=$(curl -s -X POST $BASE/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{"email":"student@test.com","password":"Test1234!"}' \
  | python3 -c "import sys,json;print(json.load(sys.stdin)['accessToken'])")

# 2. Create a payment intent for a course
curl -s -X POST $BASE/api/enrollments/intent \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer $STUDENT_TOKEN" \
  -d '{"courseId":"<published-course-id>"}' | python3 -m json.tool

# 3. Check enrollment (before payment)
curl -s $BASE/api/enrollments/check/<course-id> \
  -H "Authorization: Bearer $STUDENT_TOKEN"
# Expected: { "enrolled": false }

# 4. Simulate a webhook (use Stripe CLI: stripe trigger payment_intent.succeeded)

# 5. Check enrollment (after payment)
curl -s $BASE/api/enrollments/check/<course-id> \
  -H "Authorization: Bearer $STUDENT_TOKEN"
# Expected: { "enrolled": true }

# 6. List enrollments
curl -s $BASE/api/enrollments/mine \
  -H "Authorization: Bearer $STUDENT_TOKEN" | python3 -m json.tool
```
