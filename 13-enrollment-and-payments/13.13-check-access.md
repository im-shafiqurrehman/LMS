# Check Enrollment Access

GET /api/enrollments/check/:courseId — returns { enrolled: true/false } for the authenticated student.

Used by the course detail page (chapter 14) to decide whether to show "Enroll Now" or "Start Learning".

```js
async function checkAccess(studentId, courseId) {
  const enrollment = await Enrollment.findOne({ studentId, courseId });
  return { enrolled: !!enrollment };
}
```

✅ Returns { enrolled: true } if the student is enrolled.
✅ Returns { enrolled: false } if not.
✅ Always returns 200 (not 403) — this is a query, not a protected resource access.
