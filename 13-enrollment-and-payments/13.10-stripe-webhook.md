# The Stripe Webhook Handler

POST /api/enrollments/webhook — receives payment confirmations from Stripe.

```js
import Stripe from 'stripe';
const stripe = new Stripe(config.stripeSecretKey);

enrollmentsRouter.post('/webhook', async (req, res) => {
  const sig    = req.headers['stripe-signature'];
  let event;
  try {
    event = stripe.webhooks.constructEvent(req.body, sig, config.stripeWebhookSecret);
  } catch (err) {
    return res.status(400).json({ error: `Webhook error: ${err.message}` });
  }

  if (event.type === 'payment_intent.succeeded') {
    const intent = event.data.object;
    await enrollmentsService.confirmEnrollment(intent);
  }

  res.json({ received: true });
});
```

Always respond with 200 quickly. Stripe retries if the server does not respond within a few seconds. Move heavy processing to a background job or call it with .catch(console.error) and return immediately if needed.
