# Scaffold the Enrollment Module

The most critical configuration: the webhook endpoint needs the raw (unparsed) request body. Express's express.json() middleware parses it into a JavaScript object — but Stripe's signature verification requires the original bytes.

In app.js, add a raw body middleware only for the webhook route before express.json():

```js
// app.js — order matters
app.use('/api/enrollments/webhook', express.raw({ type: 'application/json' }));
app.use(express.json()); // All other routes get parsed JSON

// Mount routers
app.use('/api/enrollments', enrollmentsRouter);
```

The webhook route handler must be registered at /api/enrollments/webhook specifically, and it will receive a Buffer (not a parsed object) as req.body.
