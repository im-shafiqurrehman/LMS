# Download Certificate Endpoint

GET /api/learn/certificates/:enrollmentId — returns the certificate with pdfUrl for download.

```js
async function getCertificate(studentId, enrollmentId) {
  const enrollment = await Enrollment.findById(enrollmentId);
  if (!enrollment || enrollment.studentId.toString() !== studentId) {
    throw new AppError('Certificate not found', 404);
  }
  const cert = await Certificate.findOne({ enrollmentId });
  if (!cert) throw new AppError('Certificate not issued yet', 404);
  return cert;
}
```

✅ Returns 404 if the student tries to fetch another student's certificate.
✅ Returns 404 if the certificate has not been issued yet.
