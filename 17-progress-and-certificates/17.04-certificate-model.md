# Certificate Model (Revisited)

Certificates are issued once per enrollment — the unique: true on enrollmentId prevents duplicates.

For the student dashboard, you want to know quickly which courses have certificates. A single query:

```js
const certs = await Certificate.find({ enrollmentId: { $in: enrollmentIds } })
  .select('enrollmentId pdfUrl issuedAt');
```

Then map by enrollmentId for O(1) lookup when rendering each course card.
