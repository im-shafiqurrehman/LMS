# Test Progress and Certificates

```bash
BASE=http://localhost:3000
# (token from enrolled student login)

# Get progress for a course
curl -s $BASE/api/learn/<courseId>/progress \
  -H 'Authorization: Bearer <token>' | python3 -m json.tool

# Mark all lessons complete (repeat for each lessonId)
curl -s -X POST $BASE/api/learn/progress \
  -H 'Content-Type: application/json' \
  -H 'Authorization: Bearer <token>' \
  -d '{"courseId":"<id>","lessonId":"<id>"}'

# Issue certificate (after all lessons complete)
curl -s -X POST $BASE/api/learn/certificates/<courseId> \
  -H 'Authorization: Bearer <token>' | python3 -m json.tool

# List certificates
curl -s $BASE/api/learn/certificates \
  -H 'Authorization: Bearer <token>' | python3 -m json.tool
```

✅ After marking all lessons complete, POST /certificates returns a certificate.
✅ isComplete: true in the progress response.
