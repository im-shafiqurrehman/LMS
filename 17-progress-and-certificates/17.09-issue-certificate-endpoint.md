# Certificate Issuance Endpoint (Refined)

The auto-issue on completion (in markLessonComplete) means the POST /api/learn/certificates/:courseId endpoint is now idempotent: it either finds or creates the certificate.

Update it to also trigger PDF generation if pdfUrl is null:

```js
async function issueCertificate(studentId, courseId) {
  const enrollment = await requireEnrollment(studentId, courseId);
  
  const [totalLessons, completedCount] = await Promise.all([
    Lesson.countDocuments({ courseId }),
    ProgressRecord.countDocuments({ enrollmentId: enrollment._id }),
  ]);
  if (completedCount < totalLessons) throw new AppError('Complete all lessons first', 422);

  const cert = await Certificate.findOneAndUpdate(
    { enrollmentId: enrollment._id },
    { issuedAt: new Date() },
    { upsert: true, new: true }
  );

  // TODO: trigger PDF generation if cert.pdfUrl is null
  return cert;
}
```
