# What You Will Build

Admin API endpoints:
- GET /api/admin/courses/pending — list courses awaiting review
- PATCH /api/admin/courses/:id/approve — approve a course (status → published)
- PATCH /api/admin/courses/:id/reject — reject with feedback
- GET /api/admin/metrics — platform metrics
- GET /api/admin/users — user list with pagination
- PATCH /api/admin/users/:id/role — change a user's role

Admin frontend:
- /dashboard/admin — pending courses queue and metrics
