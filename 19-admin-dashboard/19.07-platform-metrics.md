# Platform Metrics

GET /api/admin/metrics — returns counts for the metrics dashboard.

```js
async function getPlatformMetrics() {
  const [totalUsers, totalCourses, totalEnrollments, revenue] = await Promise.all([
    User.countDocuments({}),
    Course.countDocuments({ status: 'published' }),
    Enrollment.countDocuments({}),
    Enrollment.aggregate([{ $group: { _id: null, total: { $sum: '$amountPaid' } } }]),
  ]);

  return {
    totalUsers,
    publishedCourses: totalCourses,
    totalEnrollments,
    totalRevenue:    revenue[0]?.total || 0,
  };
}
```

Four queries in parallel. The revenue aggregation uses amountPaid (the price snapshot) — this is accurate regardless of later price changes.
