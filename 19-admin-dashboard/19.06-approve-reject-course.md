# Approve and Reject Courses

Approve:
```js
async function approveCourse(courseId) {
  const course = await Course.findByIdAndUpdate(
    courseId, { status: 'published' }, { new: true }
  );
  if (!course) throw new AppError('Course not found', 404);
  await invalidateCatalogueCache(); // from chapter 18
  return course;
}
```

Reject (with feedback stored in the document):
```js
async function rejectCourse(courseId, { rejectionReason }) {
  const course = await Course.findByIdAndUpdate(
    courseId,
    { status: 'rejected', rejectionReason },
    { new: true }
  );
  if (!course) throw new AppError('Course not found', 404);
  return course;
}
```

Add rejectionReason to the Course schema: { type: String, default: null }.

The instructor sees this reason when they view their rejected course in the dashboard.
