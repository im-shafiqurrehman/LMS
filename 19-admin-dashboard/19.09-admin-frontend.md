# Admin Frontend

Create app/dashboard/admin/page.tsx with three tabs: Pending Courses, Metrics, Users.

The admin dashboard uses the same layout pattern as the instructor dashboard: AuthGuard + RoleGuard(admin).

Pending courses tab:
```tsx
{pending.map(c => (
  <div key={c.id}>
    <h2>{c.title}</h2>
    <p>by {c.instructorId.email}</p>
    <button onClick={() => approve(c.id)}>Approve</button>
    <button onClick={() => reject(c.id)}>Reject</button>
  </div>
))}
```

Metrics tab:
```tsx
<div>
  <p>Total users: {metrics.totalUsers}</p>
  <p>Published courses: {metrics.publishedCourses}</p>
  <p>Total enrollments: {metrics.totalEnrollments}</p>
  <p>Total revenue: ${metrics.totalRevenue.toFixed(2)}</p>
</div>
```
