# User Management

GET /api/admin/users — paginated user list:
```js
async function listUsers({ cursor, limit = 20 }) {
  const filter = cursor ? { _id: { $lt: cursor } } : {};
  const users  = await User.find(filter).sort({ _id: -1 }).limit(limit + 1);
  const hasMore    = users.length > limit;
  return { users: users.slice(0, limit), nextCursor: hasMore ? users[limit - 1]._id : null, hasMore };
}
```

PATCH /api/admin/users/:id/role — change user role:
```js
async function changeUserRole(userId, { role }) {
  return User.findByIdAndUpdate(userId, { role }, { new: true });
}
```

✅ Admin can promote a student to instructor.
✅ Admin can demote an instructor to student.
❌ Admin cannot change their own role (prevent accidental self-demotion — add a guard).
