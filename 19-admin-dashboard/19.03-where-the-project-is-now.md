# Where the Project Is Now

Create the module scaffold:
```bash
mkdir server/src/modules/admin
touch server/src/modules/admin/admin.routes.js
touch server/src/modules/admin/admin.service.js
```

Mount in app.js with admin-only access:
```js
import { adminRouter } from './modules/admin/admin.routes.js';
app.use('/api/admin', authenticate, authorize('admin'), adminRouter);
```
