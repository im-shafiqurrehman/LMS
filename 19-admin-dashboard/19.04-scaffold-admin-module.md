# Admin Module Scaffold

The admin module is simpler than others — fewer business rules, more data visibility. The admin can see and change anything.

```js
// admin.routes.js — shape only
import { Router }       from 'express';
import { adminService } from './admin.service.js';

export const adminRouter = Router();

adminRouter.get('/courses/pending',         /* list pending */ );
adminRouter.patch('/courses/:id/approve',   /* approve */ );
adminRouter.patch('/courses/:id/reject',    /* reject */ );
adminRouter.get('/metrics',                 /* platform metrics */ );
adminRouter.get('/users',                   /* user list */ );
adminRouter.patch('/users/:id/role',        /* change role */ );
```
