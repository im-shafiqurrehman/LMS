# What You Will Build

Pages and components for the instructor dashboard:

- /dashboard/instructor — list of the instructor's courses
- /dashboard/instructor/courses/new — create course form
- /dashboard/instructor/courses/[id] — edit course + lesson editor
- CourseForm component (shared between new and edit)
- LessonEditor component

State management: a dedicated instructorSlice in Redux for the instructor's courses, or local state in the page (acceptable for the instructor dashboard — only one instructor is looking at their own data at a time).
