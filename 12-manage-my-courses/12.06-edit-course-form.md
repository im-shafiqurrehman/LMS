# Edit Course Form

Create app/dashboard/instructor/courses/[id]/page.tsx. It loads the course, shows an edit form, and manages the lesson editor.

The page fetches the course on mount, populates the form fields, and saves changes via PATCH /api/instructor/courses/:id. This is a large page — break it into smaller components as you build: CourseDetailsForm and LessonEditor as separate components rendered inside the page.
