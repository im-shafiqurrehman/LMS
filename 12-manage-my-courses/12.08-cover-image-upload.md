# Cover Image Upload (URL Input)

The cover image is currently a URL input (the stub from chapter 11). Add it to the edit course form:

```tsx
// In the edit course page
const [coverUrl, setCoverUrl] = useState(course?.coverImageUrl || '');

async function handleCoverSave() {
  await api.post(`/api/instructor/courses/${courseId}/cover`, { coverImageUrl: coverUrl });
}

// In the form:
<input
  value={coverUrl}
  onChange={e => setCoverUrl(e.target.value)}
  placeholder="Cover image URL (https://...)"
/>
<button onClick={handleCoverSave}>Save Cover</button>
```

The bar-raiser version of this feature uses a file input that uploads to Cloudinary directly from the browser and posts the resulting URL. This is a good enhancement to attempt after the core flow works.
