# Choose Your Bar-Raiser

A bar-raiser is a feature you add that goes beyond the requirements — something that shows you can think like a senior developer, not just follow instructions.

Choose one:

**Technical:**
- Signed S3 URLs for lesson videos (replaces public videoUrl)
- PDF certificate generation with pdfkit
- WebSocket real-time notifications when a course is approved
- Stripe connect for instructor payouts (revenue splitting)

**Architecture:**
- Extract course catalogue into a separate read service (CQRS pattern)
- Add a message queue (BullMQ + Redis) for background jobs (PDF generation, email sending)
- Add Jest + Supertest API integration tests

**Frontend:**
- Dark mode with CSS variables
- Course search with URL state (shareable filtered URLs)
- Instructor analytics dashboard (enrollment trends, revenue chart)

Pick one, implement it, and document it in your ADR.
