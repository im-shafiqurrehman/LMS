# Your Portfolio Story

EduFlow is not just a project — it is a story you tell in job interviews. Here is the framework:

**The problem:** Existing LMS platforms are expensive for individual instructors. EduFlow gives instructors a self-hosted, extensible platform.

**The technical challenges you solved:**
- Two-token auth pattern with Redis-stored, hashed refresh tokens
- Stripe webhook idempotency preventing duplicate enrollments
- Cursor pagination for the course catalogue
- Redis cache with course-publish invalidation
- RBAC middleware + ownership checks preventing IDOR

**The decisions you made and why:**
- MongoDB for flexible course schema + horizontal scale
- Mongoose for schema validation and pre-save hooks
- Redux Toolkit for predictable state (not React Query — one source of truth)
- Express + Next.js separation for API independence

For each decision: state what the alternatives were, why you chose what you did, and what you would do differently at 10x scale.
