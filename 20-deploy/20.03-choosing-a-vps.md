# Choosing a VPS

For EduFlow, a $6/month VPS (1 vCPU, 1GB RAM) is sufficient for a portfolio project with hundreds of users.

Providers (all have similar pricing):
- **Hetzner** (CX11, Berlin/Finland): cheapest for EU
- **DigitalOcean** (Basic Droplet, NYC/London): excellent documentation
- **Vultr** (Cloud Compute, global): good performance
- **Linode/Akamai**: reliable and battle-tested

Pick any one. The server setup is identical.

Create an Ubuntu 22.04 LTS server. Add your SSH public key during creation. Connect: ssh root@your-server-ip
