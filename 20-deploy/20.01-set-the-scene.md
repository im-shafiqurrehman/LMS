# Deploy — Set the Scene

EduFlow is feature-complete. Now it needs to run at a public HTTPS URL with real secrets, not localhost. This chapter deploys both the Express API and the Next.js frontend to production.

The deployment target: a VPS (Virtual Private Server) — a Linux machine you control, running on DigitalOcean, Hetzner, Vultr, or any cloud provider. This is not serverless or managed — it is a real server you configure yourself. That is intentional: you should understand what the platform does before you abstract it away.
