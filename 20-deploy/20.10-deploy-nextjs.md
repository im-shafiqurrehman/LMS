# Deploy Next.js

The Next.js build creates an optimized production bundle in .next/:

```bash
cd /home/deploy/eduflow/client
npm run build     # ~ 30-60 seconds
npm start         # or via PM2: pm2 start npm --name eduflow-client -- start
```

PM2 serves the Next.js app on port 3001 (or whatever PORT is set in .env.local). Nginx proxies non-API requests to this port.

For the Next.js .env.local on the server:
```
NEXT_PUBLIC_API_URL=https://your-domain.com
NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY=pk_live_your_live_key
```

IMPORTANT: NEXT_PUBLIC_* variables are baked into the JavaScript bundle at build time. You must rebuild the Next.js app after changing them.
