# SSL with Certbot

```bash
apt install -y certbot python3-certbot-nginx
certbot --nginx -d your-domain.com -d www.your-domain.com
```

Certbot automatically:
1. Obtains a Let's Encrypt certificate
2. Updates the Nginx config with SSL settings
3. Sets up automatic renewal

After certbot runs, visit https://your-domain.com — you should see the Next.js homepage with a valid SSL certificate.

✅ Test SSL: curl -I https://your-domain.com — HTTP/2 200 in the response headers.
✅ Test redirect: curl -I http://your-domain.com — 301 redirect to HTTPS.
