# Environment Secrets on the Server

On the server, create /home/deploy/eduflow/server/.env with production values:

```
NODE_ENV=production
PORT=3000
DATABASE_URL=mongodb+srv://user:password@cluster.mongodb.net/eduflow
JWT_SECRET=<minimum-64-char-production-secret>
JWT_REFRESH_SECRET=<different-minimum-64-char-production-secret>
FRONTEND_URL=https://your-domain.com
STRIPE_SECRET_KEY=sk_live_your_live_key
STRIPE_WEBHOOK_SECRET=whsec_your_live_webhook_secret
REDIS_URL=redis://localhost:6379
```

This file is on the server's filesystem — never in Git.

Generate secure secrets:
```bash
node -e "console.log(require('crypto').randomBytes(64).toString('hex'))"
```

✅ Run this twice — once for JWT_SECRET, once for JWT_REFRESH_SECRET.
