# What You Will Build

- A configured Linux server with Node.js, Nginx, and PM2
- MongoDB Atlas as the production database
- The Express API running as a PM2 process
- The Next.js app built and served via PM2
- Nginx as the reverse proxy (SSL termination, static files)
- Let's Encrypt SSL certificate via Certbot
- GitHub Actions CI/CD workflow (bar-raiser)

Final state: https://your-domain.com serves the Next.js frontend; https://your-domain.com/api/* is proxied to the Express API.
