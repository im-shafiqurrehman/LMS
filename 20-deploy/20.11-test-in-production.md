# Test in Production

After deployment, run through the critical flows:

1. Visit https://your-domain.com — homepage loads with SSL
2. Register a new account — OTP is logged to PM2 logs
3. Log in — redirected to dashboard
4. Browse courses — catalogue loads from the database
5. Enroll in a course (Stripe test mode in production if using pk_test_* keys)
6. Watch a lesson — video loads from the URL
7. Mark a lesson complete
8. Check PM2 logs for any errors: pm2 logs

✅ All flows work at the HTTPS domain.
✅ No console errors in the browser.
✅ PM2 logs show no uncaught errors.
