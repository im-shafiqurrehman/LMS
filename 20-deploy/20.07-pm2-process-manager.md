# PM2 Process Manager

PM2 keeps your Node.js processes running, restarts them on crash, and starts them automatically after a server reboot.

Start the Express API:
```bash
cd /home/deploy/eduflow/server
pm2 start src/server.js --name eduflow-api
pm2 save             # save the current process list
pm2 startup          # configure PM2 to start on boot
```

Build and start the Next.js frontend:
```bash
cd /home/deploy/eduflow/client
npm run build        # Next.js production build
pm2 start npm --name eduflow-client -- start
pm2 save
```

Verify:
```bash
pm2 list            # both processes should show status: online
pm2 logs            # tail combined logs from all processes
```
