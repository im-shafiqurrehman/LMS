# Where the Project Is Now

Before you start building, orient yourself. Here is what exists:

- server/src/models/ has User and RefreshToken (from chapter 05)
- server/src/middleware/ has errorHandler.js and validate.js
- server/src/modules/ is empty
- server/src/config/index.js has JWT_SECRET and JWT_REFRESH_SECRET validated

What you are about to add:

```
server/
└── src/
    └── modules/
        └── auth/
            ├── auth.routes.js      ← HTTP routes, wired to service functions
            ├── auth.service.js     ← business logic: DB queries, token operations
            ├── auth.validators.js  ← Zod schemas for request bodies
            └── auth.helpers.js     ← token generation, OTP generation
```

After this chapter, auth.routes.js is mounted in app.js at /api/auth.

Install the auth dependencies now:

```
cd server
npm install jsonwebtoken bcrypt
npm install --save-dev @types/jsonwebtoken
```

(bcrypt was installed in chapter 05 for the User model hook — skip if already present.)
