# Logout

Logout does three things: calls POST /api/auth/logout, clears sessionStorage, and dispatches the logout action.

```ts
// A reusable logout function — use in Navbar or any page
async function handleLogout() {
  const refreshToken = sessionStorage.getItem('refreshToken');
  if (refreshToken) {
    await api.post('/api/auth/logout', { refreshToken }).catch(() => {});
    sessionStorage.removeItem('refreshToken');
  }
  store.dispatch(logout());
  router.push('/login');
}
```

The catch(() => {}) silences errors if the refresh token is already invalid — the logout should succeed regardless.

Add a Logout button to the Navbar (which you will build in 07.11):

```tsx
<button onClick={handleLogout}>Log out</button>
```

✅ Clicking logout navigates to /login and the Redux store shows isAuthenticated: false.
✅ Refreshing the page after logout keeps the user logged out.
