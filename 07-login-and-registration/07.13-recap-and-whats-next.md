# Recap and What Is Next

Here is what this chapter added:

- store/authSlice.ts — manages user, accessToken, isAuthenticated
- lib/api.ts — axios instance with auth header and 401 interceptor
- app/register/page.tsx — registration form
- app/verify/page.tsx — OTP verification
- app/login/page.tsx — login form, dispatches setCredentials
- Providers.tsx — auth state rehydration on page load
- AuthGuard component — redirects unauthenticated users
- Navbar.tsx — shows correct links based on auth state

What this chapter did NOT build:
- Role-based access control (chapter 08)
- Course catalogue (chapter 09–10)
- Any payment or enrollment UI

**What is next:** Chapter 08 — Authorization. The backend needs an authorize middleware that checks user roles before allowing access to instructor and admin endpoints.
