# The API Service — lib/api.ts

Implement the axios instance that all API calls flow through.

```ts
// lib/api.ts
import axios from 'axios';
import { store } from '@/store/store';
import { logout } from '@/store/authSlice';

const api = axios.create({
  baseURL: process.env.NEXT_PUBLIC_API_URL,
  headers: { 'Content-Type': 'application/json' },
});

// Attach access token to every request
api.interceptors.request.use((config) => {
  const token = store.getState().auth.accessToken;
  if (token) {
    config.headers.Authorization = `Bearer ${token}`;
  }
  return config;
});

// On 401, dispatch logout so the UI reflects the expired session
api.interceptors.response.use(
  (response) => response,
  (error) => {
    if (error.response?.status === 401) {
      store.dispatch(logout());
    }
    return Promise.reject(error);
  }
);

export default api;
```

Every API call in EduFlow uses this instance. The request interceptor adds the auth header automatically. The response interceptor clears the auth state on 401 — so a page that gets a 401 response immediately sees isAuthenticated: false.

✅ Do this: import api from '@/lib/api' everywhere.
❌ Do not use fetch() directly — it bypasses the interceptors and the auth header will not be set.
