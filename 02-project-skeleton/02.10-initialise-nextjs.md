# Initialise Next.js

From the project root (not inside server/):

```
npx create-next-app@latest client --typescript --tailwind --eslint --app
```

When the prompts appear:
- Would you like to use the src/ directory? → No
- Would you like to use the App Router? → Yes  
- Would you like to customise the default import alias? → No

This creates client/ with TypeScript, Tailwind CSS, ESLint, and the App Router configured.

**Add .gitignore inside client/:**

```
# client/.gitignore
.next
node_modules
.env.local
.env*.local
```

✅ .env.local is in .gitignore before you create it.
❌ Do not commit .env.local. NEXT_PUBLIC_API_URL is not a secret, but the habit must hold.

Test that it boots:

```
cd client
npm run dev -- --port 3001
```

You should see the default Next.js welcome page at http://localhost:3001.
