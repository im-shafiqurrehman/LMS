# Connect the Frontend to the Backend

Three files wire the frontend to the backend: the Redux store, the Redux Provider, and the axios instance.

**store/store.ts** — the Redux store skeleton (you will add more slices in later chapters):

```ts
import { configureStore } from '@reduxjs/toolkit';
import { TypedUseSelectorHook, useDispatch, useSelector } from 'react-redux';
import authReducer   from './authSlice';
import courseReducer from './courseSlice';

export const store = configureStore({
  reducer: {
    auth:    authReducer,
    courses: courseReducer,
  },
});

export type RootState   = ReturnType<typeof store.getState>;
export type AppDispatch = typeof store.dispatch;

export const useAppDispatch: () => AppDispatch        = useDispatch;
export const useAppSelector: TypedUseSelectorHook<RootState> = useSelector;
```

**components/Providers.tsx** — wraps the Redux Provider (must be a client component):

```tsx
'use client';
import { Provider } from 'react-redux';
import { store }    from '@/store/store';

export function Providers({ children }: { children: React.ReactNode }) {
  return <Provider store={store}>{children}</Provider>;
}
```

**app/layout.tsx** — use the provider in the root layout:

```tsx
import { Providers } from '@/components/Providers';
import './globals.css';

export default function RootLayout({ children }: { children: React.ReactNode }) {
  return (
    <html lang="en">
      <body>
        <Providers>{children}</Providers>
      </body>
    </html>
  );
}
```

✅ Do this: import { Providers } — named export from Providers.tsx.
❌ Do not import store directly in page components — always use useAppSelector and useAppDispatch.
