# Install Frontend Dependencies

```
cd client
npm install axios @reduxjs/toolkit react-redux
```

That is the complete dependency list for state management and API calls. Nothing else.

**Why no React Query or SWR?**

React Query and SWR are excellent tools for server-state caching — they shine when you need to refetch data on focus, deduplicate concurrent requests, or manage complex cache keys. EduFlow's state needs are simpler: who is logged in, and what courses are loaded. Redux Toolkit handles both cleanly.

Adding a data-fetching library on top of Redux would mean two sources of truth for the same data — the Redux store and the React Query cache. That duplication creates bugs that are difficult to trace. For EduFlow: Redux Toolkit + axios only.

✅ Verify: @reduxjs/toolkit, react-redux, and axios appear in client/package.json dependencies.
❌ Do not install react-query, @tanstack/react-query, swr, zustand, or jotai.
