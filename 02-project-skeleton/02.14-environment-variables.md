# Environment Variables

Both apps need environment files. Create them now and confirm they are in .gitignore.

**server/.env:**

```
NODE_ENV=development
PORT=3000

DATABASE_URL=mongodb://admin:yourpassword@localhost:27017/eduflow?authSource=admin

JWT_SECRET=replace-with-a-minimum-32-character-secret-string
JWT_REFRESH_SECRET=replace-with-a-different-32-character-secret

FRONTEND_URL=http://localhost:3001
```

Leave Stripe, Cloudinary, Redis, and Resend blank for now — you will fill them in when those chapters arrive.

**client/.env.local:**

```
NEXT_PUBLIC_API_URL=http://localhost:3000
```

**Confirm .gitignore coverage:**

```
# From server/
grep ".env" .gitignore

# From client/
grep ".env.local" .gitignore
```

Both should match. Then verify git status shows no .env files staged.

✅ .env exists only on your local machine — never in git log.
✅ The server reads NEXT_PUBLIC_API_URL from client/.env.local, not hardcoded.
❌ Never hardcode http://localhost:3000 in source code — it will break in production.
