# Initialise the Backend

In the server/ directory, initialise the Node project and add Git hygiene before anything else:

```
cd server
npm init -y
git init  # if not already done at the root level
```

Create .gitignore immediately — before npm install, before .env:

```
# server/.gitignore
node_modules
.env
.env.*
dist
```

✅ Do this before npm install. If node_modules is already on disk, git add . would stage it without a .gitignore.
❌ Never skip the .gitignore step. "I'll add it later" is how secrets end up in git log.

Commit this state:

```
git add .gitignore package.json
git commit -m "chore(server): initialise Node project"
```

The project skeleton starts with a clean commit history — one commit per meaningful change, not "WIP" blobs at the end of the day.
