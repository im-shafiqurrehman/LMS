# Install Backend Dependencies

```
cd server
npm install express cors helmet morgan mongoose zod
npm install --save-dev nodemon
```

**What each package does:**

| Package | Role |
|---------|------|
| express | The HTTP framework. Handles routing, middleware, request/response. |
| cors | Allows the Next.js frontend (different origin) to make requests to the API. |
| helmet | Sets security headers (X-Frame-Options, Content-Security-Policy, etc.) on every response. One line of protection. |
| morgan | Logs every incoming request to stdout. You will thank yourself when debugging at 2am. |
| mongoose | The MongoDB ODM. Defines schemas, validates documents, exposes a query API. |
| zod | Schema validation library. Used for env config validation at startup and request body validation at runtime. |
| nodemon | Dev-only: watches for file changes and restarts the server automatically. |

Add scripts to package.json:

```json
"scripts": {
  "start": "node src/server.js",
  "dev":   "nodemon src/server.js"
}
```

✅ Verify: node_modules/ exists and mongoose appears in package.json dependencies.
❌ Do not install jsonwebtoken, bcrypt, or other auth packages yet — those come in chapter 06.
