# Delete a Lesson

DELETE /api/instructor/courses/:id/lessons/:lid — removes a lesson from the course.

```js
async function deleteLesson(courseId, lessonId, userId, role) {
  const course = await Course.findById(courseId);
  if (!course) throw new AppError('Course not found', 404);
  if (role !== 'admin' && course.instructorId.toString() !== userId) {
    throw new AppError('Forbidden', 403);
  }

  const result = await Lesson.findOneAndDelete({ _id: lessonId, courseId });
  if (!result) throw new AppError('Lesson not found', 404);

  await syncCourseTotals(courseId);
  return { message: 'Lesson deleted' };
}
```

After deletion, positions are not re-numbered — this is acceptable for a draft course. The frontend lesson editor displays lessons sorted by position; gaps in the sequence are invisible to the end user.

✅ course.totalLessons decrements after delete.
✅ Deleting a lesson that does not belong to the course returns 404.
