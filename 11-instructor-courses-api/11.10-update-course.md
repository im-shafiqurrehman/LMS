# Update Course

PATCH /api/instructor/courses/:id — updates course metadata. Only allowed on draft courses.

```js
const updateCourseSchema = z.object({
  title:       z.string().min(5).max(200).optional(),
  description: z.string().min(20).optional(),
  price:       z.coerce.number().min(0).max(9999).optional(),
  categoryId:  z.string().optional(),
});
```

Business rules enforced in the service:
- Owner or admin only
- Cannot update a pending, published, or rejected course (only admins can un-reject)

```js
async function updateCourse(courseId, userId, role, updates) {
  const course = await Course.findById(courseId);
  if (!course) throw new AppError('Course not found', 404);
  if (role !== 'admin' && course.instructorId.toString() !== userId) throw new AppError('Forbidden', 403);
  if (course.status !== 'draft') throw new AppError('Can only edit draft courses', 422);
  Object.assign(course, updates);
  return course.save();
}
```
