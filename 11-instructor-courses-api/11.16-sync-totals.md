# Sync Course Totals

After every lesson add/update/delete, totalLessons and totalDuration on the Course document must be recalculated. This syncCourseTotals helper uses a MongoDB aggregation:

```js
// instructor.service.js — helper
async function syncCourseTotals(courseId) {
  const result = await Lesson.aggregate([
    { $match: { courseId: new mongoose.Types.ObjectId(courseId) } },
    { $group: {
      _id:           null,
      totalLessons:  { $sum: 1 },
      totalDuration: { $sum: '$durationSeconds' },
    }},
  ]);

  const totals = result[0] || { totalLessons: 0, totalDuration: 0 };
  await Course.findByIdAndUpdate(courseId, {
    totalLessons:  totals.totalLessons,
    totalDuration: totals.totalDuration,
  });
}
```

One aggregation query replaces any manual incrementing/decrementing — which accumulates errors if a delete or update is missed. The aggregation recomputes from the ground truth (the actual lesson documents).

✅ After adding 3 lessons with durationSeconds 600, 900, 1200: course.totalDuration === 2700.
✅ After deleting one 600s lesson: course.totalDuration === 2100.
