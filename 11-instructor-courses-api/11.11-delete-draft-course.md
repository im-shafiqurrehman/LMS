# Delete Draft Course

DELETE /api/instructor/courses/:id — deletes a draft course and all its lessons.

Business rules:
- Owner only (no admin bypass — only the instructor can delete their own course)
- Only draft courses can be deleted (prevents deleting published courses with active enrollments)

```js
async function deleteCourse(courseId, userId) {
  const course = await Course.findById(courseId);
  if (!course) throw new AppError('Course not found', 404);
  if (course.instructorId.toString() !== userId) throw new AppError('Forbidden', 403);
  if (course.status !== 'draft') throw new AppError('Only draft courses can be deleted', 422);

  await Lesson.deleteMany({ courseId });
  await course.deleteOne();
  return { message: 'Course deleted' };
}
```

✅ DELETE on own draft course succeeds; course and its lessons are removed.
✅ DELETE on a published course returns 422.
✅ DELETE on another instructor's course returns 403.
✅ After delete, the course is not returned by GET /api/instructor/courses/mine.
