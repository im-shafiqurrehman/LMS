# Chapter 11 — Definition of Done

- [ ] POST /api/instructor/courses creates a draft course
- [ ] GET /api/instructor/courses/mine returns only the instructor's own courses
- [ ] PATCH /api/instructor/courses/:id updates metadata on draft courses only
- [ ] DELETE /api/instructor/courses/:id deletes the course and all its lessons
- [ ] POST /api/instructor/courses/:id/lessons adds a lesson; totalLessons updated
- [ ] PATCH and DELETE on lessons work; totalDuration kept in sync via aggregation
- [ ] PATCH /api/instructor/courses/:id/submit changes status to 'pending'; fails if no lessons
- [ ] Student token on any instructor route returns 403
- [ ] Instructor A cannot edit Instructor B's course (403)
- [ ] git commit: "feat(instructor): CRUD courses, lessons, submit for review"
