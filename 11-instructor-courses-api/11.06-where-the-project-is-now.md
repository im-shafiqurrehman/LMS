# Where the Project Is Now

Create the module scaffold:

```bash
mkdir server/src/modules/instructor
touch server/src/modules/instructor/instructor.routes.js
touch server/src/modules/instructor/instructor.service.js
touch server/src/modules/instructor/instructor.validators.js
```

Mount in app.js with the appropriate middleware:

```js
import { instructorRouter } from './modules/instructor/instructor.routes.js';
app.use('/api/instructor', authenticate, authorize('instructor', 'admin'), instructorRouter);
```

By applying authenticate and authorize at the router level, every route in the instructor module is automatically protected. No need to add the middleware to each individual route.

Restart and verify:
```
curl http://localhost:3000/api/instructor/courses
```
Expected: 401 (no token) — the router is mounted and protected.
