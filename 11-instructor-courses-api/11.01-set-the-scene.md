# Instructor Courses API — Set the Scene

Instructors are the content creators of EduFlow. This chapter gives them the tools to do their job: create a course, write the description, add lessons, upload a cover image, and submit for admin review. Only the admin can publish a course.

The instructor experience has six states:
- draft: just created, instructor is still editing
- pending: submitted for review
- published: admin approved, visible in the catalogue
- rejected: admin rejected with feedback

This chapter builds the backend. Chapter 12 builds the frontend.

By the end:
- An instructor can create, update, and delete draft courses (their own only)
- An instructor can add, update, and delete lessons (within their own courses)
- An instructor can submit a course for review
- totalLessons and totalDuration are kept in sync automatically
