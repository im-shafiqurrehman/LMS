# Create Draft Course

POST /api/instructor/courses — creates a new course with status: 'draft'.

Validators:
```js
const createCourseSchema = z.object({
  title:       z.string().min(5).max(200),
  description: z.string().min(20),
  price:       z.coerce.number().min(0).max(9999),
  categoryId:  z.string().optional(),
});
```

Service:
```js
async function createCourse(instructorId, data) {
  const slug = slugify(data.title, { lower: true, strict: true });
  return Course.create({
    instructorId,
    slug,
    status: 'draft',
    ...data,
  });
}
```

Install slugify: npm install slugify

✅ POST with valid data returns 201 with the new course document.
✅ POST without a title returns 422 with field-level errors.
✅ The created course has status: 'draft'.
