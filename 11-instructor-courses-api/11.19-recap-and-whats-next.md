# Recap and What Is Next

This chapter added the complete instructor course management API:

- Create, update, delete draft courses
- Add, update, delete lessons (with automatic totalLessons/totalDuration sync via aggregation)
- Submit for review (status: draft → pending)
- Cover image URL stub

Ownership checks prevent instructors from modifying each other's courses. The authorize middleware prevents students from accessing any of these routes.

**What is next:** Chapter 12 — Manage My Courses (Frontend). The instructor dashboard in Next.js — a list of the instructor's courses, create/edit forms, lesson editor, and submit button.
