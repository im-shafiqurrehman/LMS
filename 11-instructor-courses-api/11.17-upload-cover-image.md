# Upload Cover Image (Stub)

POST /api/instructor/courses/:id/cover — updates the course's cover image URL.

For now, implement as a stub that accepts a URL string directly. Real file uploads (Cloudinary or S3 signed URL) can be added as a bar-raiser feature.

```js
const coverSchema = z.object({
  coverImageUrl: z.string().url(),
});

instructorRouter.post('/:id/cover', validate(coverSchema), async (req, res, next) => {
  try {
    const course = await Course.findById(req.params.id);
    if (!course) return res.status(404).json({ error: 'Course not found' });
    if (course.instructorId.toString() !== req.user.id) {
      return res.status(403).json({ error: 'Forbidden' });
    }
    course.coverImageUrl = req.body.coverImageUrl;
    await course.save();
    res.json(course);
  } catch (err) { next(err); }
});
```

The frontend course editor will show an input for the image URL until real file upload is wired.

✅ Passing a valid HTTPS image URL updates coverImageUrl on the course.
✅ Passing an invalid URL returns 422.
