# List My Courses

GET /api/instructor/courses/mine — returns all courses belonging to the authenticated instructor.

```js
async function listMyCourses(instructorId) {
  return Course.find({ instructorId })
    .populate('categoryId', 'name slug')
    .sort({ createdAt: -1 });
}
```

✅ As an instructor, GET /api/instructor/courses/mine returns only your courses (not other instructors').
✅ The response includes the category name for each course.
✅ As a student, GET /api/instructor/courses/mine returns 403.
