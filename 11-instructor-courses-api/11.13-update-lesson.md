# Update a Lesson

PATCH /api/instructor/courses/:id/lessons/:lid — updates a lesson within the instructor's course.

```js
async function updateLesson(courseId, lessonId, userId, role, updates) {
  const course = await Course.findById(courseId);
  if (!course) throw new AppError('Course not found', 404);
  if (role !== 'admin' && course.instructorId.toString() !== userId) {
    throw new AppError('Forbidden', 403);
  }

  const lesson = await Lesson.findOneAndUpdate(
    { _id: lessonId, courseId },
    updates,
    { new: true, runValidators: true }
  );
  if (!lesson) throw new AppError('Lesson not found', 404);

  await syncCourseTotals(courseId);
  return lesson;
}
```

Note { runValidators: true } — without it, Mongoose skips validation on updates. Always include it for findOneAndUpdate calls.

✅ Updating durationSeconds recalculates course.totalDuration.
✅ Updating a lesson that does not belong to the course returns 404.
