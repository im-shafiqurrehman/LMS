# Test the Instructor API with curl

Login as an instructor and test the full workflow:

```bash
BASE=http://localhost:3000

# 1. Login as instructor
INSTRUCTOR_TOKEN=$(curl -s -X POST $BASE/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{"email":"instructor@test.com","password":"Test1234!"}' \
  | python3 -c "import sys,json;print(json.load(sys.stdin)['accessToken'])")

# 2. Create a draft course
COURSE=$(curl -s -X POST $BASE/api/instructor/courses \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer $INSTRUCTOR_TOKEN" \
  -d '{"title":"My First Course","description":"Learn things in this twenty-plus character description","price":49}')

COURSE_ID=$(echo $COURSE | python3 -c "import sys,json;print(json.load(sys.stdin)['id'])")
echo "Course ID: $COURSE_ID"

# 3. Add a lesson
curl -s -X POST $BASE/api/instructor/courses/$COURSE_ID/lessons \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer $INSTRUCTOR_TOKEN" \
  -d '{"title":"Introduction","durationSeconds":600}'

# 4. Submit for review
curl -s -X PATCH $BASE/api/instructor/courses/$COURSE_ID/submit \
  -H "Authorization: Bearer $INSTRUCTOR_TOKEN"

# 5. Verify status is pending
curl -s $BASE/api/instructor/courses/mine \
  -H "Authorization: Bearer $INSTRUCTOR_TOKEN" \
  | python3 -c "import sys,json;[print(c['title'],c['status']) for c in json.load(sys.stdin)]"
```

✅ Each step returns the expected result.
✅ After submit, course status is 'pending'.
✅ Student token on any instructor route returns 403.
