# The Lesson Model (Revisited)

The Lesson schema was defined in chapter 05. Before building the instructor module, verify it is correct:

```js
{
  courseId:        ObjectId ref Course,
  title:           String required,
  videoUrl:        String default null,
  durationSeconds: Number default 0,
  position:        Number required
}
```

The position field is the instructor-controlled ordering of lessons within a course. When the instructor adds a new lesson, position defaults to lessons.length + 1 (append to end). Reordering is a bar-raiser feature — for now, new lessons are appended.

After any lesson is added, updated, or deleted, the course's totalLessons and totalDuration must be recalculated. This is done by the syncCourseTotals() helper in the service.
