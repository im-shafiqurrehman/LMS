# Add a Lesson

POST /api/instructor/courses/:id/lessons — adds a lesson to a draft course.

```js
const addLessonSchema = z.object({
  title:           z.string().min(3).max(200),
  videoUrl:        z.string().url().optional(),
  durationSeconds: z.coerce.number().min(0).default(0),
});

async function addLesson(courseId, userId, role, lessonData) {
  const course = await Course.findById(courseId);
  if (!course) throw new AppError('Course not found', 404);
  if (role !== 'admin' && course.instructorId.toString() !== userId) {
    throw new AppError('Forbidden', 403);
  }
  if (course.status !== 'draft') throw new AppError('Can only add lessons to draft courses', 422);

  const existingCount = await Lesson.countDocuments({ courseId });
  const lesson = await Lesson.create({
    courseId,
    position: existingCount + 1,
    ...lessonData,
  });

  await syncCourseTotals(courseId);
  return lesson;
}
```

After adding a lesson, syncCourseTotals recalculates course.totalLessons and course.totalDuration.

✅ POST with a valid lesson returns 201 with the lesson document.
✅ course.totalLessons increments after each add.
