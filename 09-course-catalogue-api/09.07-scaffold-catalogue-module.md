# Scaffold the Catalogue Module

The catalogue module has only read-only routes. No authentication required.

```js
// src/modules/catalogue/catalogue.routes.js — shape only
import { Router }           from 'express';
import { catalogueService } from './catalogue.service.js';

export const catalogueRouter = Router();

catalogueRouter.get('/',       async (req, res, next) => { /* list courses */ });
catalogueRouter.get('/:id',    async (req, res, next) => { /* single course */ });
catalogueRouter.get('/:id/lessons', async (req, res, next) => { /* lesson list */ });
```

The service is a plain object of async functions. Keep the routes thin — route handlers extract req.query / req.params, call service, and return the result:

```js
// src/modules/catalogue/catalogue.service.js — shape only
export const catalogueService = {
  async listCourses({ categoryId, q, cursor, limit }) { ... },
  async getCourseById(id) { ... },
  async getCourseLessons(courseId) { ... },
};
```

Route handlers are max 10 lines — the complexity lives in the service.
