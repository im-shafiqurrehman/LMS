# Where the Project Is Now

Checklist before you start building:

- The server is running (npm run dev in server/)
- MongoDB is running (docker compose ps shows mongodb as running)
- The seed data from chapter 05 is in the database (two published courses)
- The Course model has indexes on { status, createdAt } and { categoryId, status }

Create the module scaffold:

```
mkdir server/src/modules/catalogue
touch server/src/modules/catalogue/catalogue.routes.js
touch server/src/modules/catalogue/catalogue.service.js
touch server/src/modules/catalogue/catalogue.validators.js
```

Then mount the router in app.js:

```js
import { catalogueRouter } from './modules/catalogue/catalogue.routes.js';
app.use('/api/courses', catalogueRouter);
```

Restart and verify the mount works:
```
curl http://localhost:3000/api/courses
```
Expected: 404 (no route defined yet) — confirms the path is reachable.
