# GET Course Lessons Endpoint

The GET /api/courses/:id/lessons endpoint returns the full lesson list for a course. Same as the lesson preview in getCourseById, but as a standalone route for the lesson player.

```js
// catalogue.service.js — getCourseLessons
async getCourseLessons(courseId) {
  const course = await Course.findById(courseId).select('status');
  if (!course || course.status !== 'published') {
    throw new AppError('Course not found', 404);
  }

  return Lesson
    .find({ courseId })
    .select('title durationSeconds position')
    .sort({ position: 1 });
}
```

Route:
```js
catalogueRouter.get('/:id/lessons', async (req, res, next) => {
  try {
    const lessons = await catalogueService.getCourseLessons(req.params.id);
    res.json(lessons);
  } catch (err) {
    next(err);
  }
});
```

Video URLs are still excluded. They are only accessible via the lesson access API after enrollment verification (chapter 15).
