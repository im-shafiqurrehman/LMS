# Test the Catalogue API with curl

```bash
BASE=http://localhost:3000

# 1. List all published courses
curl -s "$BASE/api/courses" | python3 -m json.tool

# 2. Filter by category
curl -s "$BASE/api/courses?categoryId=<category-id>" | python3 -m json.tool

# 3. Keyword search
curl -s "$BASE/api/courses?q=javascript" | python3 -m json.tool

# 4. Pagination — first page
PAGE1=$(curl -s "$BASE/api/courses?limit=3")
echo $PAGE1 | python3 -m json.tool
CURSOR=$(echo $PAGE1 | python3 -c "import sys,json;print(json.load(sys.stdin)['nextCursor'])")

# 5. Pagination — second page using cursor
curl -s "$BASE/api/courses?limit=3&cursor=$CURSOR" | python3 -m json.tool

# 6. Single course detail
curl -s "$BASE/api/courses/<course-id>" | python3 -m json.tool
# Verify: no videoUrl on lessons

# 7. Course lessons
curl -s "$BASE/api/courses/<course-id>/lessons" | python3 -m json.tool
```

✅ All six commands return expected results.
✅ Lesson list in course detail has no videoUrl field.
✅ nextCursor on the first page returns a different set of courses on the second page.
