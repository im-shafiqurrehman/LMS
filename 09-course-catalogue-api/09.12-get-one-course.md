# Get a Single Course

GET /api/courses/:id returns the full course detail — title, description, price, instructor info, lesson count, and lesson list (titles and durations only, no video URLs).

```js
// catalogue.service.js — getCourseById
async getCourseById(id) {
  const course = await Course
    .findById(id)
    .populate('instructorId', 'email')
    .populate('categoryId',   'name slug');

  if (!course) throw new AppError('Course not found', 404);
  if (course.status !== 'published') throw new AppError('Course not found', 404);

  const lessons = await Lesson
    .find({ courseId: id })
    .select('title durationSeconds position')  // NO videoUrl
    .sort({ position: 1 });

  return { course, lessons };
}
```

Video URLs are intentionally excluded from the catalogue — students can only access video URLs if they are enrolled (chapter 15).

✅ GET /api/courses/:id returns the course with lessons (no videoUrl on each lesson).
✅ GET /api/courses/:id for a draft course returns 404 — not a 403. Do not leak that the course exists.
