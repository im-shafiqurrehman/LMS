# Seed More Test Data

The seed from chapter 05 created two published courses. The catalogue needs more variety to test filtering and pagination.

Update server/src/lib/seed.js to add:
- 3 courses in category "Web Development"
- 3 courses in category "Data Science"
- 1 draft course (should not appear in the public catalogue)
- 1 pending course (should not appear in the public catalogue)
- Each course with 3–5 lessons

After updating the seed script:

```
npm run seed
```

Then verify:

```
curl http://localhost:3000/api/courses
# Expected: up to 6 published courses (not the draft or pending ones)

curl "http://localhost:3000/api/courses?categoryId=<web-dev-id>"
# Expected: 3 web development courses only

curl "http://localhost:3000/api/courses?q=python"
# Expected: data science courses (if you used "Python" in the title)
```

✅ 6 published courses in total, 0 draft or pending courses in the response.
✅ Category filter works correctly.
✅ Keyword search works.
