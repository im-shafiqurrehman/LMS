# Lesson List Component

Extract the lesson list into a reusable component: components/LessonList.tsx.

```tsx
// components/LessonList.tsx
import { Lesson } from '@/types';

export function LessonList({ lessons }: { lessons: Lesson[] }) {
  return (
    <ul style={{ listStyle: 'none', padding: 0 }}>
      {lessons.map((lesson, index) => (
        <li
          key={lesson.id}
          style={{
            display:        'flex',
            justifyContent: 'space-between',
            padding:        '0.75rem 0',
            borderBottom:   '1px solid #f0f0f0',
          }}
        >
          <span>
            <span style={{ color: '#999', marginRight: '0.5rem' }}>{index + 1}.</span>
            {lesson.title}
          </span>
          <span style={{ color: '#999', fontSize: '0.875rem' }}>
            {Math.round(lesson.durationSeconds / 60)}m
          </span>
        </li>
      ))}
    </ul>
  );
}
```

This component shows titles and durations only. When a student is enrolled, the lesson items will link to the lesson player (chapter 16). The check for enrollment status before showing links is added in that chapter.
