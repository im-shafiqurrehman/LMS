# Load More Button

The catalogue uses cursor pagination. A "Load More" button requests the next page and appends the results to the existing list.

```tsx
// In CoursesPage — add below the course grid
{hasMore && (
  <button
    onClick={() => fetchCourses({ reset: false })}
    disabled={loading}
    style={{ marginTop: '1rem', display: 'block', marginLeft: 'auto', marginRight: 'auto' }}
  >
    {loading ? 'Loading...' : 'Load More'}
  </button>
)}

{!hasMore && courses.length > 0 && (
  <p style={{ textAlign: 'center', color: '#999', marginTop: '1rem' }}>
    You have seen all courses.
  </p>
)}
```

The appendCourses action (not setCourses) is dispatched when loading more — it adds to the existing list rather than replacing it.

✅ Clicking Load More appends the next set of courses to the grid.
✅ When hasMore: false, the Load More button is hidden and the "all courses" message appears.
✅ Changing a filter while on page 2 resets to page 1 (fetchCourses({ reset: true })).
