# Browse Courses — Set the Scene

The catalogue API is complete. This chapter wires it to the Next.js frontend: a courses listing page with category filters and keyword search, a course detail page, and a lesson preview list. No authentication required — these are public pages.

State management uses a courseSlice in Redux for the catalogue data. Filter state (selected category, search query, cursor) is local state in the page component.

By the end:
- /courses shows a paginated course grid with filter and search
- /courses/[id] shows full course details and a lesson list
- A "Load More" button appends the next page to the existing list
