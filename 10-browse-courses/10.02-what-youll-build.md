# What You Will Build

Three frontend pieces:

**1. course Redux slice** (store/courseSlice.ts)
Stores: catalogue courses array, loading flag, error, nextCursor, hasMore.
Actions: setCourses, appendCourses (for "load more"), setLoading, setError.

**2. /courses page** (app/courses/page.tsx)
- Fetches from GET /api/courses on mount
- Renders a grid of CourseCard components
- Has a CategoryFilter sidebar/bar
- Has a keyword search input
- Has a Load More button (disabled when hasMore: false)

**3. /courses/[id] page** (app/courses/[id]/page.tsx)
- Fetches from GET /api/courses/:id
- Renders the course detail: cover, description, instructor, price, lesson list
- Shows an Enroll button (placeholder — wired in chapter 14)
